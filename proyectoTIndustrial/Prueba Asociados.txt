							VLAN


- Cree las vlan en el servidor vtp o servidor de vlans (si hay varios switches)

		vlan __
			name Nativa

- Configure los puertos ya sean troncales o de acceso

		Troncal:         switchport mode trunk
				 switchport trunk native vlan __

		Acceso:		switchport mode access
             			 switchport access vlan __


- En la interface LAN o serial del router que se comunica con las vlan:

		int f0/0.11
			encapsulation dot1Q 11 native  -- vlan nativa
			exit
		
		int f0/0.30
			encapsulation dot1Q 30
			ip add 172.16.30.1 255.255.255.0   -- ip del router

						Spanning Tree


- Utilizado para evitar loops y caidas de la red por publicaciones de broadcast
- Ademas asegura los caminos mï¿½s rapidos y levanta paths por si los principales caen


- Modo: 
		spanning-tree mode rapid-pvst

- Root primario o secundario de vlan's:
		
		spanning-tree vlan ___ root primary/secondary


- Portfast (solo en puertos de acceso):

		spanning-tree portfast

- Comandos de seguridad:

		switchport port-security
  		switchport port-security maximum [1-132]
 		switchport port-security mac-address sticky
 		switchport port-security violation shutdown
